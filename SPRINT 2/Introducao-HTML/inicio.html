HTML básico - códigos HTML

Aprenda os princípios básicos para confecção de um site em HTML para que você possa fazer o seu primeiro site.

Motivação

HTML (abreviação de Hypertext Markup Language) é uma liguagem de marcação utilizada na estruturação de páginas web. Sua sintaxe é bastante e, assim como XML é baseada em tags, que representam os diversos elementos de uma página, como imagens e links.
